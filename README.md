# Entry-way-Ticket-System

🎟️ EntryWay Ticket – MERN Stack Project
📌 Project Overview

EntryWay Ticket is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
The application allows users to book entry tickets online, manage bookings, and securely authenticate users.

This project demonstrates full-stack development skills, including API development, database management, authentication, and frontend UI development.
